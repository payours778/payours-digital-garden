# 博客项目

一个使用 Next.js 构建的个人博客网站，包含以下页面：

- **首页** - 网站概览
- **随笔** - 文章列表与详情
- **音乐** - 音乐播放器与播放列表
- **照片墙** - 照片展示
- **时光线** - 时间线展示
- **项目** - 作品展示
- **关于** - 个人介绍
- **愿望树** - 愿望清单

## 技术栈

**前端**
- Next.js 16.2.6 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4

**后端** (开发中)
- Express.js
- SQLite
